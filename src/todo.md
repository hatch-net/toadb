# 内存上下文的管理
通过内存上下文，对动态申请的内存进行自动管理，避免内存的泄漏，同时也增加了内存边界的校验。
在每个阶段使用独立的内存上下文，在退出阶段时销毁内存上下文，就可以释放此阶段所有动态申请的内存；

对于全局存在的动态内存，在顶层内存上下文中进行分配；
